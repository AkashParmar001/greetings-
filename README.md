# Dictionary I/O Examples

This folder contains a small example showing how to get dictionary input and output in Python.

Files:
- dict_io_examples.py â€” interactive, CLI, and JSON file examples

Quick run:

```bash
python dict_io_examples.py --demo
```

Build a dict from items on the command line and save it:

```bash
python dict_io_examples.py --items name=Alice age=30 --file out.json
```

Interactively enter pairs:

```bash
python dict_io_examples.py --interactive
# then type lines like: key=value
```
